ブラウザからOVOを設定する

説明

このページからお手持ちのOVOの設定を変更することができます。
イコライザをはじめ、本体側ボタンでは選択できない追加機能もこちらで設定可能です。ぜひご活用ください。

ご利用になる前に

この設定ページをご利用頂くには Web MIDI機能を搭載したブラウザ が必要です。現時点では Chrome でのみ動作を確認しております。
また、必ずOVOのファームウェアを最新の状態に更新してからご利用ください。
詳しい対応環境や注意点につきましては 下記 をご参照ください。

各種設定

LOCAL VOLUME

OFF: OVOのボリュームレバーは接続先機器のボリュームを変更します。
ON: OVOのボリュームレバーはOVO内部のボリュームを変更します。
ONにすると100%を超えるボリューム設定も可能になりますが、音割れを引き起こす可能性がありますのでご注意ください。

HIGH BOOST、イコライザとも同時使用できます。

HIGH BOOST

BASS BOOST、イコライザとも同時使用できます。

L/R SETTING

左右のチャンネルの割り当てを選択します。
「左のみ」「右のみ」は指定した側の音声を両チャンネルにコピーして出力します。PC一台にOVOを二台以上接続する際にご利用ください。
なおL/R別イコライザはこのオプションの効果の後に適用されます。


OFF

LOW POWER

低消費電力モードで動作します。LEDも消灯します。
スマートフォンなど供給電力が小さい機器に接続して使用するためのモードです。

PLAYER CONTROL

ONにするとOVOのLEDレバーにリモートコントロール操作（Prev, Play, Next）を割り当てます。
ただし再生機器側もリモートコントロールに対応している必要があります。

OVO Pro専用設定

アナログミックス機能

OFF: アナログ接続中はAudio（Digital）端子は給電にのみ使用します。
ON: アナログ音声とAudio（Digital）端子のUSBオーディオ音声をミックスして再生します。
ONのときOVO本体のボリュームレバーはアナログ音声にのみ働きますので、USBオーディオのボリュームはUSBホスト機器側で調整してください。

リミッター

オーディオチップ内蔵リミッターを有効化します。EQ、ボリュームの後段に設置されており、初期設定はOFFです。
ボリューム適用後の振幅が-0.5dB付近に到達した瞬間のみ働きます。アタック、リリースタイムも調整可能です。
大音量再生時やアナログミックス時のひずみを軽減するにはON、デジタル出力の正確性を重視する場合はOFFにしてください。


アタック：50μs

リリース：300ms

DCオフセット除去フィルタ

低域ノイズ除去用ハイパスフィルタのカットオフ周波数を変更します。
EQの前段に設置されており、初期設定は30Hzで常時作動しています。
再生品質を保つためには初期値のまま使用することをおすすめしますが、デジタル出力の正確性を重視する場合はお好みで調整してください。

注意事項

必要なOVOのバージョン

OVOとの通信を行うためには、OVOのファームウェアを バージョン 5 以上 にアップデートして頂く必要がございます。
また一部の追加機能はより新しいバージョンのファームウェアを必要とします。必ずOVOを最新バージョンにアップデートした上でご利用ください。
アップデート方法につきましては、製品ページの ファームウェア更新 の案内をご参照ください。

対応ブラウザ

ブラウザとOVOとの通信にはWeb MIDIを使用しておりますので、 Web MIDI機能に対応したブラウザ が必要となります。
現時点では Chrome と Windows 10、または Mac（High Sierra以降）の組み合わせでのみ動作を確認しております。
（その他、非公式ですがOpera、Vivaldiでも動作する可能性がございます）
それ以外の主流ブラウザ（Firefox、Safari、IE、Edge等）はWeb MIDI機能を搭載していないためご使用になれません。ご了承ください。

なおスマートフォン版Chromeの動作は端末によって異なりますため、現時点ではサポート対象外としております。

うまく通信できない場合

「接続済み」と表示されているにもかかわらず設定がOVO本体に反映されない場合は、お手数ですが一度ページの再読み込みをお試しください。
なお、アナログ入力モードではMIDI通信を行うことができません。必ずデジタル入力モードでご利用ください。
